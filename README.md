# pipsi-wrapper

A simple wrapper to use the [pipsi](https://github.com/mitsuhiko/pipsi)

## Install

With [fisherman]

```
fisher robsonpeixoto/pipsi-wrapper
```

## Usage

```fish
pipsi-wrapper
```
